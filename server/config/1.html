技术周刊 2019-05-13：Flutter 使命升级，Facebook 洗心革面

\n
\n

写于 2019.05.13

\n
\n

前端快爆

\n
    \n
  • Flutter 近期动作频频， 发布了 1.5 版本，提供了 Android 和 iOS 的内购插件，升级到了 iOS 12.1 SDK 以响应苹果的政策……但最激动人心的莫过于 Flutter for web 的发布，这使得 Flutter 程序可以直接运行在任何浏览器上(Demo)，同时 Flutter 团队把自己的使命扩展成“打造在任何屏幕上创造美好体验的最佳框架”。🔗
    • \n
\n
\n

点评：此时的 Flutter 亮出了自己的全部野心。

\n
\n
    \n
  • \n

    Google I/O 2019 召开，Chrome 开发者团队在 YouTube 上新建了合集 Web at Google I/O 2019，囊括了此次大会有关 Web 技术的所有讲座视频，值得一看。🔗

    \n
    • \n
  • \n

    Chrome 74 发布，其中包括：类私有字段的使用、用 prefers-reduced-motion媒体查询减少动效的使用、transition events 的支持。🔗

    \n
    • \n
  • \n

    Chrome 计划于 Picture-in-Picture V2 中支持弹出任意元素，这样网站将能把网页做得更像桌面程序，当前的 Picture-in-Picture V1 仅支持视频元素的弹出。🔗

    \n
    • \n
  • \n

    在 Microsoft Build 2019 上 Edge 浏览器宣布了一些新功能，其中包括能收藏网页信息的 Collections、保护用户隐私的 Privacy tools、与 Chrome 同源的开发者工具以及 IE 模式。🔗

    \n
    • \n
\n
\n

点评： 最终，iPhone 添加了双卡双待，Edge 活成了国产双核浏览器的样子。

\n
\n
    \n
  • VS Code 新添加 Remote Development 特性，使得开发者可以在远程环境下开发、调试。🔗
    • \n
\n
\n

点评：又少了一个用 Vim 的理由。

\n
\n

全民目击

\n

去年深陷各种官司的 Facebook 在 F8 大会上宣布要“洗心革面”，进行网站和 App 的大改版。全新的设计语言被称为 FB5，在 FB5 中，FB 的经典蓝更多地被白色取代，主要信息更加突出。从使用率最高的导航 Tab 可以看出群组和市场是 Facebook 接下来关注的重心。🔗
